# Manager Web (Vue) 开发规则

## 前端最佳实践
- 使用组件化架构
- 使用ESLint和Prettier保持代码一致性
- 环境特定设置放在环境变量中
- 实现响应式设计和可访问性标准

## 安全考虑
- 保护API调用使用安全令牌
- 不在客户端代码中存储机密信息
- 实现CSRF保护
- 验证所有用户输入

## UI/UX指南
- 遵循设计稿: https://codesign.qq.com/app/s/526108506410828
- 使用Vue 3的组合式API
- 使用Vue Router进行路由管理
- 使用状态管理库（如Pinia）
